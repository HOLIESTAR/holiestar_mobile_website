小米/紅米 手機 已讀不回助理 設定

由於各廠牌使用的不同 [客製效能優化] 或 [省電] 軟體，可能因此會導致 已讀不回助理 無法正常運作，大家只要按照教學逐步設定就可以順暢使用 已讀不回助理 囉。請使用 小米/紅米 手機的 已讀不回助理 用戶嘗試以下步驟設定，現在就跟著一起操作看看吧！

  • 開始步驟

小提醒：

（1）此教學圖片來自 紅米 Note 4 機型，如果你使用的是其他 小米/紅米 機型請參考下面項目自行找尋對應設定哦

（2）重新開機或是長時間待機後，請記得檢查設定有沒有跑掉喔

步驟一

點選 [安全中心] → [授權管理] → 選擇 [自啟動管理] → 把已讀不回助理 [開啟]

步驟二

點選 [安全中心] → [授權管理] → [其他權限管理] → 選擇 [已讀不回助理] → 設定如下

步驟三

點選 [安全中心] → [電量] → 右上角[設定] → 設定如下

步驟四

點選 [安全中心] → [電量] → 右上角[設定] → 設定如下

步驟五

點選 [多工鍵] → 將[已讀不回助理]往下拖曳 → [鎖定任務]

（鎖頭出來才表示 已讀不回助理 不會被 小米/紅米 的系統清除喔）

步驟六

[設定] → [通知和狀態欄] → [通知管理] → [已讀不回助理] → 設定如下

（鎖頭出來才表示 已讀不回助理 不會被 小米/紅米 的系統清除喔）

（👉記得重複以上步驟，設定您的訊息APP(Facebook, LINE, Instagram...)，已免訊息APP被系統清除，影響已讀不回助理的運作唷👈）
